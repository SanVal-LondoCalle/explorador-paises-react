# Explorador de Países del mundo

Aplicación web creada en react.js por Valeria Calle y Santiago Londoño, esta aplicación web permite buscar y visualizar información sobre países utilizando la API pública https://restcountries.com

El usuario puede buscar por nombre o filtrar por región, y ver información detallada como bandera, capital, población y región.

---

## Características

- Busca países por nombre
- Filtrar países por región
- Visualizar:
- Nombre oficial
- Bandera
- Población
- Región / CONTINENTE
- indicador de carga
- Diseño responsivo  con Bootstrap 5
- Uso de `useState` sin `useEffect`

## Tecnologías

- React.js (componentes funcionales)
- Bootstrap 5
- Fetch API
- Rest Countries API
- Vite
